我的第一篇技术笔记

你好，这是我的博客！

欢迎来到我的个人博客空间。这是我使用 Hexo 搭建博客后写的第一篇正式文章。

为什么搭建这个博客

我一直想有一个自己的空间，用来：

  • 📝 记录学习过程中的笔记
  • 💡 分享解决问题的经验
  • 📚 整理技术文档和资料
  • 🎯 追踪自己的成长轨迹

未来计划

接下来我打算在这里分享：

  1. 学习笔记 - 编程、框架、工具的使用心得
  2. 问题总结 - 遇到的坑和解决方案
  3. 项目记录 - 个人项目的开发过程
  4. 生活随笔 - 偶尔也会写点生活感悟

结语

搭建博客的过程本身也学到了很多，从 Hexo 初始化到 GitHub Pages 部署，每一步都是新的体验。
